---
title: "Delete a file"
api: "DELETE https://api.uploadfly.cloud/delete"
description: "This endpoint deletes a single file."
---

### Body

<ParamField body="file_url" type="string">
  The file URL you provide is used to identify a file that exists in your fly
  that you want to delete.
</ParamField>

### Response

<ResponseField name="success" type="number">
  Indicates whether the call was successful. 1 if successful, 0 if not.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request DELETE 'https://api.mintlify.com/api/user' \
--header 'Content-Type: application/json' \
--header 'Authorization: Token <token>' \
--data-raw '{
		"user_group_id": "example_1"
		"current_token": "abcdef"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": 1
}
```

</ResponseExample>
